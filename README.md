# pages-zh-cn

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定域名或具体网站。

## 项目简介

这是一个通用型 HTML 页面归档仓库。仓库中的每个页面文件均为独立完整的 HTML 文档，可直接在浏览器中打开查看。仓库本身不提供统一的站点结构或导航系统，所有页面以原始文件形式存放。

## 目录说明

```
/
├── README.md          # 本文件
├── index.html         # 可选：仓库入口页（如有）
└── pages/             # 页面归档目录（如有）
    ├── page-1.html
    ├── page-2.html
    └── ...
```

- 仓库根目录下可放置一个 `index.html` 作为可选入口页面。
- `pages/` 目录（若存在）用于存放归档的 HTML 页面文件，文件名应具有辨识度且不含歧义。
- 各页面文件应保持独立，避免相互引用或依赖外部资源（除非资源已内联）。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，推荐将 CSS、JavaScript 和图片等资源内联至文件中，以确保离线可查看。
- 页面内容应符合一般网络内容规范，不包含违法、侵权或不当信息。
- 归档页面不设自动索引，访问者需通过直接链接或手动浏览目录查看。
- 仓库不提供页面列表或分类索引（除非在 `index.html` 中自行实现）。

## 维护说明

- 欢迎提交 Pull Request 新增或更新页面文件。
- 新增页面请放置在合适的目录下，并确保文件名不与其他文件冲突。
- 提交前请检查页面文件格式正确、资源内联完整、无外部依赖。
- 仓库维护者保留对页面内容进行审核和合并的权利。
- 不建议在页面中包含动态加载外部资源或追踪代码。
